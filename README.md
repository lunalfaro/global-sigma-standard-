**Open Framework for Certification of High-Risk AI Systems**

Consolidating EU AI Act, NIST RMF, OECD, UNESCO, and ISO/IEC into 12 operationalized principles (P1-P12) with reproducible audit methodology.

## ✨ Key Innovation

> Distinguishes **Technical Conformity** (voluntary, no immutability required) from **Certification** (mandatory for T3/T4, requires immutability for forensic proof).

## 🚀 Quick Start

1. **Classify your system** using Risk Tier logic (T1-T4).
2. **Self-assess** with the Annex A tool.
3. **Identify Vetos** - Remediable vs. Irremediable.
4. **Seek Certification** (if Tier 3/4 in regulated jurisdictions).
