# Global Sigma v3.1 - Open Framework for Certification of High-Risk AI Systems

**Status: Official Research Validation Edition (2026-2027)**

## 📌 Overview
**Global Sigma** is a consolidated, operationalized, and auditable certification framework for High-Risk AI systems. It translates 12 fundamental principles (P1-P12) derived from the **EU AI Act, OECD, UNESCO, NIST, and ISO/IEC** into **verifiable, measurable requirements**.

> "It distinguishes Technical Conformity (voluntary) from Certification (mandatory for T3/T4, requiring immutability for forensic proof)." [cite: 5]

## 📂 Repository Structure
- `/standard` - The official standard (PDF).
- `/tools` - Self-assessment tools (Excel/Notion templates).
- `/annexes` - Supporting documentation (Immutability patterns, Veto logic).

## 🚀 Getting Started
1. **Read the Executive Summary** (EN/ES).
2. **Classify your system** using the Risk Tier logic (T1-T4).
3. **Self-assess** against P1-P12 using the Annex A Matrix.
4. **Identify Vetos** - Determine if they are Remediable or Irremediable.
